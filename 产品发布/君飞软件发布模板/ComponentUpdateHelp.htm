普方PFCloud V5中小企业云管理软件 环境检测

1. 介绍

普方PFCloud V5中小企业云管理软件要求在安装之前，先安装基本的第三方组件。通过“普方PFCloud V5中小企业云管理软件-环境检测”程序，可以自动检测系统需要安装的组件。

2. 要求的基准组件

・                 Windows Server 2003 Service SP 2

・                 Microsoft .NET Framework 4.0

・                MSDE 2000 SP4

・                Microsoft SQL Server 2000 Service Pack 4

 

 

3. 支持的操作系统

普方PFCloud V5中小企业云管理V5.1客户端支持下列操作系统                                    

・                 Windows XP Professional SP3 x86 简体中文版

・                 Windows Server 2003 SP2 x86 x64 简体中文版

・                 Windows Server 2008 x86 x64 简体中文版

・                 Windows Vista 旗舰版 x86 x64 简体中文版

・                 Windows 7 旗舰版 x86 x64 简体中文版

 

普方PFCloud V5中小企业云管理V7.5服务器端支持下列操作系统                                

・                 Windows XP Professional SP3 x86 简体中文版

・                 Windows Server 2003 SP2 x86 x64 简体中文版

・                 Windows Vista 旗舰版 x86 x64 简体中文版

・                 Windows 7 旗舰版 x86 x64 简体中文版

 

 

4. 已知的安装问题

4.1. 中断安装

如果个别组件在其安装过程中挂起，则安装程序可能无法检测到此情况。请使用 CTRL+ALT+DELETE 手动关闭 V5Setup.exe 任务。请注意，在速度较慢的计算机上或通过网络安装较大的组件（如 Windows 2003 Service Pack 2）时，安装可能需要 20 分钟左右。

4.2. 在运行组件更新时 Norton AntiVirus 报告警报信息

在运行 Norton AntiVirus 时安装 Windows 2003 Service Pack 2 会出现已知的兼容性问题。建议您在运行组件更新安装程序之后再安装 Norton AntiVirus（建议：在安装普方PFCloud V5中小企业云管理软件以及进行组件更新时，关闭系统现有的全部防病毒软件）。

5. 附加问题

5.2. 卸载基准组件

卸载普方PFCloud V5中小企业云管理软件时不会卸载由组件更新安装的组件。对于提供单独卸载的组件，请使用“控制面板”中的“添加/删除程序”进行卸载。
